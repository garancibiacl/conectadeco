# ConectaDeco

Plataforma de conectividad y servicios para el sector de decoración y diseño de interiores.

## Stack Tecnológico

- **Frontend**: React.js, TypeScript, Tailwind CSS
- **Backend**: Node.js, Express.js, PostgreSQL
- **Infraestructura**: Docker, AWS/Azure
- **Base de Datos**: PostgreSQL
- **Versionado**: Git + GitHub

## Requisitos Previos

- Node.js (v18+)
- npm o yarn
- PostgreSQL (v13+)
- Docker (opcional)

## Instalación

```bash
cd app
npm install
cp .env.example .env
npm run dev
```

## Estructura del Proyecto

- `/app` - Aplicación principal (frontend, backend, database)
- `/hitos` - Entregas por hito
- `/docs` - Documentación técnica

## Documentación

Ver [hitos/README.md](./hitos/README.md) para entregas y [docs/arquitectura.md](./docs/arquitectura.md) para diseño técnico.
